# 🌱 Smart Vertical Gardening System – Flutter & IoT

An IoT-powered Smart Vertical Gardening mobile application built with Flutter and Firebase, designed to monitor environmental conditions and automate essential farming operations for efficient vertical agriculture.

---

# 🚀 Project Overview

The Smart Vertical Gardening System combines IoT technology with mobile application development to help users monitor and manage vertical farming environments in real time.

The application connects with IoT sensors and controllers through Firebase Realtime Database, enabling automated irrigation, environmental monitoring, and smart farming decisions from a mobile device.
